# obsidian-public

This project is a personal website and public digital garden built using **Quartz v4**. It renders the public-facing portions of my Obsidian vault into a set of static web pages.

## Project Structure & Context
- **Framework:** Quartz (v4), running on Node.js.
- **Content:** All Markdown notes, projects, and images live in the `content/` directory.
- **Configuration:** Layout and website settings are controlled by `quartz.config.ts` and `quartz.layout.ts`.

## AI Assistant Guidelines
- If asked to create or update content, edit the Markdown files inside `content/`.
- If asked to modify the site's design or functionality, you will likely need to edit the Quartz TypeScript configuration or explore the `quartz/` directory.
- Keep solutions simple and respect the existing formatting.
